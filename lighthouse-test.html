🚀 Lighthouse Testing - CryptoVault Pro

📋 Instructions

This page helps you run Lighthouse performance testing on the CryptoVault Pro application.

🔧 Steps to Run Lighthouse

  1. Open Chrome DevTools (F12)
  2. Go to the "Lighthouse" tab
  3. Select categories: Performance, Accessibility, Best Practices, SEO
  4. Click "Generate report"
  5. Save the report as HTML and take a screenshot

🎯 Expected Results

  • Performance: ≥ 95
  • Accessibility: 100
  • Best Practices: ≥ 90
  • SEO: ≥ 90

📸 After Testing

Once you have the Lighthouse results:

  • Take a screenshot of the scores
  • Save it as docs/lighthouse.png
  • Replace the existing lighthouse.png file
